## APIS

### Task API

All the task api will follow the following format `{rootUrl}/api/v1/task`

| #   | Method | Path | Description    |
| --- | ------ | ---- | -------------- |
| 1   | `GET`  | `/`  | Get all tasks  |
| 2   | `POST` | `/`  | Post the tasks |

### User API

All the user api will follow the following format `{rootUrl}/api/v1/user`
